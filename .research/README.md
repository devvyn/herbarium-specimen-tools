## Why Holistic Consolidation?

**Fragmentation (anti-pattern)**:
```
herbarium-specimen-tools/.research/  ← isolated
another-project/.research/           ← isolated
meta-project/research/               ← isolated
```

**Consolidation (holistic)**:
```
~/devvyn-meta-project/research/
├── herbarium/        ← OCR insights
├── archives/         ← OCR insights
└── cross-project/    ← NATURAL CONNECTION emerges
```

Agents discover "herbarium OCR preprocessing technique works for archives too" automatically.
